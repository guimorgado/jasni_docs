## Before You Start

You'll need:

1. A Jasni account - [Sign up free](https://app.jasni.ai)
2. An API key - Get one from your [dashboard](https://app.jasni.ai/settings/api-keys)
3. A connected email account

## Send Your First Email

<Tabs>
  <Tab title="cURL">
    ```bash
    curl -X POST https://api.jasni.ai/api/v1/emails/send \
      -H "Authorization: Bearer jsk_your_api_key" \
      -H "Content-Type: application/json" \
      -d '{
        "from": "you@yourdomain.com",
        "to": "recipient@example.com",
        "subject": "Hello from Jasni!",
        "text": "This is my first email sent via Jasni API.",
        "html": "<h1>Hello!</h1><p>This is my first email sent via Jasni API.</p>"
      }'
    ```
  </Tab>
  <Tab title="JavaScript">
    ```javascript
    const response = await fetch('https://api.jasni.ai/api/v1/emails/send', {
      method: 'POST',
      headers: {
        'Authorization': 'Bearer jsk_your_api_key',
        'Content-Type': 'application/json',
      },
      body: JSON.stringify({
        from: 'you@yourdomain.com',
        to: 'recipient@example.com',
        subject: 'Hello from Jasni!',
        text: 'This is my first email sent via Jasni API.',
        html: '<h1>Hello!</h1><p>This is my first email sent via Jasni API.</p>'
      }),
    });

    const data = await response.json();
    console.log(data);
    ```
  </Tab>
  <Tab title="Python">
    ```python
    import requests

    response = requests.post(
        'https://api.jasni.ai/api/v1/emails/send',
        headers={
            'Authorization': 'Bearer jsk_your_api_key',
            'Content-Type': 'application/json',
        },
        json={
            'from': 'you@yourdomain.com',
            'to': 'recipient@example.com',
            'subject': 'Hello from Jasni!',
            'text': 'This is my first email sent via Jasni API.',
            'html': '<h1>Hello!</h1><p>This is my first email sent via Jasni API.</p>'
        }
    )

    print(response.json())
    ```
  </Tab>
</Tabs>

### Response

```json
{
  "success": true,
  "data": {
    "messageId": "<abc123@yourdomain.com>",
    "from": "you@yourdomain.com",
    "to": ["recipient@example.com"],
    "subject": "Hello from Jasni!",
    "savedToSent": true
  }
}
```

<Check>
  Congratulations! You've sent your first email with Jasni.
</Check>

## Read Your Inbox

Fetch emails from your inbox:

<Tabs>
  <Tab title="cURL">
    ```bash
    curl -X GET "https://api.jasni.ai/api/v1/emails?account=you@yourdomain.com&folder=INBOX&limit=10" \
      -H "Authorization: Bearer jsk_your_api_key"
    ```
  </Tab>
  <Tab title="JavaScript">
    ```javascript
    const response = await fetch(
      'https://api.jasni.ai/api/v1/emails?account=you@yourdomain.com&folder=INBOX&limit=10',
      {
        headers: {
          'Authorization': 'Bearer jsk_your_api_key',
        },
      }
    );

    const data = await response.json();
    console.log(data.data.emails);
    ```
  </Tab>
  <Tab title="Python">
    ```python
    import requests

    response = requests.get(
        'https://api.jasni.ai/api/v1/emails',
        headers={
            'Authorization': 'Bearer jsk_your_api_key',
        },
        params={
            'account': 'you@yourdomain.com',
            'folder': 'INBOX',
            'limit': 10
        }
    )

    emails = response.json()['data']['emails']
    for email in emails:
        print(f"From: {email['from']} - Subject: {email['subject']}")
    ```
  </Tab>
</Tabs>

## Set Up Webhooks

Receive real-time notifications when new emails arrive:

<Tabs>
  <Tab title="cURL">
    ```bash
    curl -X POST https://api.jasni.ai/api/v1/webhooks \
      -H "Authorization: Bearer jsk_your_api_key" \
      -H "Content-Type: application/json" \
      -d '{
        "url": "https://your-server.com/webhook",
        "events": ["email.received", "email.sent"],
        "description": "My first webhook"
      }'
    ```
  </Tab>
  <Tab title="JavaScript">
    ```javascript
    const response = await fetch('https://api.jasni.ai/api/v1/webhooks', {
      method: 'POST',
      headers: {
        'Authorization': 'Bearer jsk_your_api_key',
        'Content-Type': 'application/json',
      },
      body: JSON.stringify({
        url: 'https://your-server.com/webhook',
        events: ['email.received', 'email.sent'],
        description: 'My first webhook'
      }),
    });

    const data = await response.json();
    // Save the secret! It's only shown once
    console.log('Webhook secret:', data.data.webhook.secret);
    ```
  </Tab>
</Tabs>

<Warning>
  **Save your webhook secret!** The `secret` is only returned when the webhook is created. You'll need it to verify incoming webhook payloads.
</Warning>

## What is Jasni?

Jasni AI is a powerful email infrastructure platform designed for developers and AI agents. Send, receive, and manage emails programmatically with our simple REST API. Perfect for building:

  <Card title="AI Email Agents" icon="robot">
    Build intelligent agents that read, understand, and respond to emails automatically
  </Card>
  <Card title="Email Automation" icon="gears">
    Automate workflows triggered by incoming emails with webhooks
  </Card>
  <Card title="Customer Support" icon="headset">
    Create smart support systems that handle inquiries 24/7
  </Card>
  <Card title="Transactional Email" icon="paper-plane">
    Send notifications, receipts, and updates reliably
  </Card>
</CardGroup>

## Quick Start

Get up and running with Jasni in minutes.

<Steps>
  <Step title="Get your API key">
    Sign up at [app.jasni.ai](https://app.jasni.ai) and create an API key from your dashboard.
  </Step>
  <Step title="Connect your email">
    Add your email account or create a new one with a custom domain.
  </Step>
  <Step title="Start building">
    Use our REST API to send, receive, and manage emails programmatically.
  </Step>
</Steps>

## Core Features

<AccordionGroup>
  <Accordion title="Email Management" icon="envelope">
    - List, read, and search emails from any folder
    - Send emails with HTML and plain text support
    - Reply to and forward emails with proper threading
    - Mark emails as read/unread, delete, and organize with labels
  </Accordion>
  <Accordion title="Real-time Webhooks" icon="bolt">
    - Receive instant notifications when emails arrive
    - Track email delivery and bounce events
    - Secure payloads with HMAC signatures
    - Automatic retries with exponential backoff
  </Accordion>
  <Accordion title="Labels & Organization" icon="tags">
    - Create custom labels to categorize emails
    - Apply labels programmatically for AI classification
    - Filter and search by labels
  </Accordion>
  <Accordion title="Security & Reliability" icon="shield-check">
    - Bearer token authentication
    - Rate limiting protection
    - HTTPS-only API endpoints
    - 99.9% uptime SLA
  </Accordion>
</AccordionGroup>

## SDKs & Integrations

<Note>
  Official SDKs are coming soon! In the meantime, you can use any HTTP client to interact with our REST API.
</Note>

```bash
# Example: Send an email with curl
curl -X POST https://api.jasni.ai/api/v1/emails/send \
  -H "Authorization: Bearer jsk_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "from": "you@yourdomain.com",
    "to": "recipient@example.com",
    "subject": "Hello from Jasni!",
    "text": "This is my first email sent via Jasni API."
  }'
```

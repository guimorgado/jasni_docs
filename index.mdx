## What is Jasni?

Jasni AI is an email infrastructure platform designed for developers and AI agents. Send, receive, and manage emails programmatically with our simple REST API.

## Quick Start

Get up and running with Jasni in minutes.

<Steps>
  <Step title="Get your API key">
    Sign up at [app.jasni.ai](https://app.jasni.ai) and create an API key from your dashboard.
  </Step>
  <Step title="Connect your email">
    Add your email account or create a new one with a custom domain.
  </Step>
  <Step title="Start building">
    Use our REST API to send, receive, and manage emails programmatically.
  </Step>
</Steps>

## Features

<AccordionGroup>
  <Accordion title="Email Management" icon="envelope">
    - List, read, and search emails from any folder
    - Send emails with HTML and plain text support

    - Mark emails as read/unread, delete, and organize with labels

## Example

Send an email with a single API call:

```bash
curl -X POST https://api.jasni.ai/api/v1/emails/send \
  -H "Authorization: Bearer jsk_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "from": "you@yourdomain.com",
    "to": "recipient@example.com",
    "subject": "Hello from Jasni!",
    "text": "This is my first email sent via Jasni API."
  }'
```

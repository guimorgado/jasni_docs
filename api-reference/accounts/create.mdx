---
title: "Create Account"
api: "POST /api/v1/accounts"
description: "Create a new email account"
---

## Request Body

<ParamField body="username" type="string">
  The local part of the email address (before @). If not provided, a random username is generated.
</ParamField>

<ParamField body="name" type="string">
  Display name for the account (optional). Defaults to the username if not provided.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the account was created successfully
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="account" type="object">
      The created account object
      <Expandable title="properties">
        <ResponseField name="id" type="string">
          Unique identifier (UUID)
        </ResponseField>
        <ResponseField name="email" type="string">
          Full email address
        </ResponseField>
        <ResponseField name="name" type="string">
          Display name
        </ResponseField>
        <ResponseField name="created_at" type="string">
          ISO 8601 creation timestamp
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://api.jasni.ai/api/v1/accounts \
  -H "Authorization: Bearer jsk_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "username": "agent-001",
    "name": "Sales Agent"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.jasni.ai/api/v1/accounts', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer jsk_your_api_key',
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    username: 'agent-001',
    name: 'Sales Agent',
  }),
});

const data = await response.json();
console.log('Created account:', data.data.account.email);
```

```python Python
import requests

response = requests.post(
    'https://api.jasni.ai/api/v1/accounts',
    headers={
        'Authorization': 'Bearer jsk_your_api_key',
        'Content-Type': 'application/json',
    },
    json={
        'username': 'agent-001',
        'name': 'Sales Agent',
    }
)

account = response.json()['data']['account']
print(f"Created: {account['email']}")
```

## Validation

### Username
- Optional (random generated if not provided)
- 3-64 characters
- Can only contain: letters, numbers, dots, dashes, underscores
- Must be unique

## Notes

- If no username is provided, a random 12-character username is generated
- The email domain is determined by your Jasni configuration
- Each created account can be used to send and receive emails

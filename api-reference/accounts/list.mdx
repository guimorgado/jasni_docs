---
title: "List Accounts"
api: "GET /api/v1/accounts"
description: "Retrieve all email accounts for the authenticated user"
---

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="accounts" type="array">
      List of email account objects
      <Expandable title="account properties">
        <ResponseField name="id" type="string">
          Unique identifier (UUID)
        </ResponseField>
        <ResponseField name="email" type="string">
          Email address
        </ResponseField>
        <ResponseField name="name" type="string">
          Display name
        </ResponseField>
        <ResponseField name="created_at" type="string">
          ISO 8601 creation timestamp
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="total" type="integer">
      Total number of accounts
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET https://api.jasni.ai/api/v1/accounts \
  -H "Authorization: Bearer jsk_your_api_key"
```

```javascript JavaScript
const response = await fetch('https://api.jasni.ai/api/v1/accounts', {
  headers: {
    'Authorization': 'Bearer jsk_your_api_key',
  },
});

const data = await response.json();
console.log(data.data.accounts);
```

```python Python
import requests

response = requests.get(
    'https://api.jasni.ai/api/v1/accounts',
    headers={'Authorization': 'Bearer jsk_your_api_key'}
)

accounts = response.json()['data']['accounts']
for account in accounts:
    print(f"{account['email']} - {account['name']}")
```
</RequestExample>

## Notes

- Returns all email accounts owned by the authenticated user
- Accounts are returned sorted by creation date (newest first)

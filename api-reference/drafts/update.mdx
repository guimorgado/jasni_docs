---
title: "Update Draft"
api: "PUT /api/v1/emails/drafts"
description: "Update an existing draft email"
---

## Request Body

<ParamField body="account" type="string" required>
  Email account
</ParamField>

<ParamField body="uid" type="integer" required>
  UID of the draft to update
</ParamField>

<ParamField body="to" type="string">
  Recipient email address
</ParamField>

<ParamField body="subject" type="string">
  Email subject
</ParamField>

<ParamField body="text" type="string">
  Plain text body
</ParamField>

<ParamField body="html" type="string">
  HTML body
</ParamField>

<ParamField body="cc" type="string">
  CC recipients
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the draft was updated successfully
</ResponseField>

<ResponseField name="uid" type="integer">
  UID of the new draft (may change after update)
</ResponseField>

<ResponseField name="account" type="string">
  The email account
</ResponseField>

<ResponseField name="message" type="string">
  Confirmation message
</ResponseField>

<RequestExample>
```bash cURL
curl -X PUT https://api.jasni.ai/api/v1/emails/drafts \
  -H "Authorization: Bearer jsk_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "account": "you@domain.com",
    "uid": 123,
    "to": "recipient@example.com",
    "subject": "Updated: Follow up on our meeting",
    "text": "Hi,\n\nUpdated draft content..."
  }'
```

```javascript JavaScript
const response = await fetch('https://api.jasni.ai/api/v1/emails/drafts', {
  method: 'PUT',
  headers: {
    'Authorization': 'Bearer jsk_your_api_key',
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    account: 'you@domain.com',
    uid: 123,
    to: 'recipient@example.com',
    subject: 'Updated: Follow up on our meeting',
    text: 'Hi,\n\nUpdated draft content...',
  }),
});

const data = await response.json();
console.log('Draft updated, new UID:', data.uid);
```

```python Python
import requests

response = requests.put(
    'https://api.jasni.ai/api/v1/emails/drafts',
    headers={
        'Authorization': 'Bearer jsk_your_api_key',
        'Content-Type': 'application/json',
    },
    json={
        'account': 'you@domain.com',
        'uid': 123,
        'to': 'recipient@example.com',
        'subject': 'Updated: Follow up on our meeting',
        'text': 'Hi,\n\nUpdated draft content...',
    }
)

print(f"Draft updated: {response.json()['uid']}")
```
</RequestExample>

<ResponseExample>
```json Success
{
  "success": true,
  "uid": 124,
  "account": "you@domain.com",
  "message": "Draft updated successfully"
}
```

```json Error - Draft not found
{
  "error": "Failed to update draft"
}
```

```json Error - Missing UID
{
  "error": "Draft UID required"
}
```
</ResponseExample>

## Notes

- Updating a draft replaces the old draft with a new one
- The UID may change after the update (IMAP limitation)
- The old draft is deleted and a new one is created

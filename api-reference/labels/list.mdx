---
title: "List Labels"
api: "GET /api/v1/labels"
description: "Retrieve all labels for the authenticated user"
---

## Request

<ParamField query="include_count" type="boolean" default="false">
  Include email count per label
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="labels" type="array">
      List of label objects
      <Expandable title="label properties">
        <ResponseField name="id" type="string">
          Unique identifier (UUID)
        </ResponseField>
        <ResponseField name="name" type="string">
          Label name
        </ResponseField>
        <ResponseField name="color" type="string">
          Hex color code
        </ResponseField>
        <ResponseField name="description" type="string">
          Optional description
        </ResponseField>
        <ResponseField name="created_at" type="string">
          ISO 8601 creation timestamp
        </ResponseField>
        <ResponseField name="email_count" type="integer">
          Number of emails (when `include_count=true`)
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="total" type="integer">
      Total number of labels
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.jasni.ai/api/v1/labels" \
  -H "Authorization: Bearer jsk_your_api_key"
```

```bash cURL - With counts
curl -X GET "https://api.jasni.ai/api/v1/labels?include_count=true" \
  -H "Authorization: Bearer jsk_your_api_key"
```

```javascript JavaScript
const response = await fetch('https://api.jasni.ai/api/v1/labels', {
  headers: {
    'Authorization': 'Bearer jsk_your_api_key',
  },
});

const data = await response.json();
console.log(data.data.labels);
```

```python Python
import requests

response = requests.get(
    'https://api.jasni.ai/api/v1/labels',
    headers={'Authorization': 'Bearer jsk_your_api_key'},
    params={'include_count': 'true'}
)

labels = response.json()['data']['labels']
for label in labels:
    print(f"{label['name']}: {label.get('email_count', 'N/A')} emails")
```

## Notes

- Labels are returned sorted by name in ascending order
- The `email_count` field is only included when `include_count=true`
- Each label has a unique UUID identifier

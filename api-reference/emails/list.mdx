---
title: "List Emails"
api: "GET /api/v1/emails"
description: "Retrieve a list of emails from a specified email account"
---

## Request

<ParamField query="account" type="string" required>
  Email account to fetch from (e.g., `user@example.com`)
</ParamField>

<ParamField query="folder" type="string" default="INBOX">
  Folder to fetch from. Common folders: `INBOX`, `Sent`, `Drafts`, `Trash`, `Spam`
</ParamField>

<ParamField query="limit" type="integer" default="50">
  Number of emails to fetch (max 100)
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="emails" type="array">
      List of email objects
    </ResponseField>
    <ResponseField name="folder" type="string">
      The folder that was queried
    </ResponseField>
    <ResponseField name="total" type="integer">
      Number of emails returned
    </ResponseField>
    <ResponseField name="account" type="string">
      The email account that was queried
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.jasni.ai/api/v1/emails?account=you@domain.com&folder=INBOX&limit=10" \
  -H "Authorization: Bearer jsk_your_api_key"
```

```javascript JavaScript
const response = await fetch(
  'https://api.jasni.ai/api/v1/emails?account=you@domain.com&folder=INBOX&limit=10',
  {
    headers: {
      'Authorization': 'Bearer jsk_your_api_key',
    },
  }
);
const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.jasni.ai/api/v1/emails',
    headers={'Authorization': 'Bearer jsk_your_api_key'},
    params={
        'account': 'you@domain.com',
        'folder': 'INBOX',
        'limit': 10
    }
)
data = response.json()
```
</RequestExample>

## Notes

- If no `account` parameter is provided, the endpoint returns a list of available email accounts instead of emails
- The `limit` parameter has a maximum value of 100
- Emails are returned in reverse chronological order (newest first)
- The `flags` array contains IMAP flags like `\Seen` (read), `\Flagged` (starred), etc.

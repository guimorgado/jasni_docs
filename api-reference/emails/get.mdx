---
title: "Get Email"
api: "GET /api/v1/emails/{id}"
description: "Retrieve a specific email by its UID"
---

## Request

<ParamField path="id" type="integer" required>
  Email UID (unique identifier within the mailbox)
</ParamField>

<ParamField query="account" type="string" required>
  Email account the email belongs to
</ParamField>

<ParamField query="folder" type="string" default="INBOX">
  Folder where the email is located
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="email" type="object">
      <Expandable title="properties">
        <ResponseField name="uid" type="integer">
          Unique identifier
        </ResponseField>
        <ResponseField name="messageId" type="string">
          RFC 2822 Message-ID
        </ResponseField>
        <ResponseField name="from" type="string">
          Sender email address
        </ResponseField>
        <ResponseField name="to" type="array">
          Recipient email addresses
        </ResponseField>
        <ResponseField name="cc" type="array">
          CC recipients
        </ResponseField>
        <ResponseField name="subject" type="string">
          Email subject
        </ResponseField>
        <ResponseField name="date" type="string">
          ISO 8601 datetime
        </ResponseField>
        <ResponseField name="text" type="string">
          Plain text body
        </ResponseField>
        <ResponseField name="html" type="string">
          HTML body
        </ResponseField>
        <ResponseField name="attachments" type="array">
          List of attachments
        </ResponseField>
        <ResponseField name="flags" type="array">
          IMAP flags
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.jasni.ai/api/v1/emails/12345?account=you@domain.com&folder=INBOX" \
  -H "Authorization: Bearer jsk_your_api_key"
```

```javascript JavaScript
const response = await fetch(
  'https://api.jasni.ai/api/v1/emails/12345?account=you@domain.com&folder=INBOX',
  {
    headers: {
      'Authorization': 'Bearer jsk_your_api_key',
    },
  }
);
const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.jasni.ai/api/v1/emails/12345',
    headers={'Authorization': 'Bearer jsk_your_api_key'},
    params={
        'account': 'you@domain.com',
        'folder': 'INBOX'
    }
)
email = response.json()['data']['email']
```
</RequestExample>

## Notes

- The `id` must be a valid integer representing the email's UID
- The `account` parameter is required to identify which mailbox to query
- The full email body (both `text` and `html`) is returned, unlike the list endpoint which may truncate

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="webhooks" type="array">
      List of webhook objects
      <Expandable title="webhook properties">
        <ResponseField name="id" type="string">
          Unique identifier (UUID)
        </ResponseField>
        <ResponseField name="url" type="string">
          Webhook endpoint URL
        </ResponseField>
        <ResponseField name="events" type="array">
          Subscribed events
        </ResponseField>
        <ResponseField name="active" type="boolean">
          Whether the webhook is enabled
        </ResponseField>
        <ResponseField name="description" type="string">
          Optional description
        </ResponseField>
        <ResponseField name="created_at" type="string">
          ISO 8601 creation timestamp
        </ResponseField>
        <ResponseField name="updated_at" type="string">
          ISO 8601 last update timestamp
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET https://api.jasni.ai/api/v1/webhooks \
  -H "Authorization: Bearer jsk_your_api_key"
```

```javascript JavaScript
const response = await fetch('https://api.jasni.ai/api/v1/webhooks', {
  headers: {
    'Authorization': 'Bearer jsk_your_api_key',
  },
});

const data = await response.json();
console.log(data.data.webhooks);
```

```python Python
import requests

response = requests.get(
    'https://api.jasni.ai/api/v1/webhooks',
    headers={'Authorization': 'Bearer jsk_your_api_key'}
)

webhooks = response.json()['data']['webhooks']
for webhook in webhooks:
    status = "✓" if webhook['active'] else "✗"
    print(f"{status} {webhook['url']} - {webhook['events']}")
```
</RequestExample>

## Notes

- Webhooks are returned in reverse chronological order (newest first)
- The `secret` field is **never** returned when listing webhooks (only on creation)
- Maximum of 10 webhooks per user

## Available Events

| Event | Description |
|-------|-------------|
| `email.received` | A new email was received |
| `email.sent` | An email was sent successfully |
| `email.delivered` | An email was delivered to the recipient's server |
| `email.bounced` | An email bounced |
| `email.spam` | A recipient marked the email as spam |
| `email.rejected` | An email was rejected before delivery |
| `email.deleted` | An email was deleted |
| `domain.verified` | A custom domain was verified |

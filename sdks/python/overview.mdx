## Installation

Install the SDK using pip:

```bash
pip install jasni
```

## Quick Start

```python
from jasni import Jasni

# Initialize the client with your API key
jasni = Jasni("jsk_your_api_key")

# List your agent's inboxes
accounts = jasni.accounts.list()
print(f"Your agent has {accounts.total} inboxes")

# Let your agent send an email
result = jasni.emails.send(
    from_="agent@yourdomain.com",
    to="user@example.com",
    subject="Your request has been processed",
    text="I have completed the task you requested. Here are the results...",
)
print(f"Agent sent email with ID: {result.message_id}")
```

## Async Support

The SDK supports async operations with `AsyncJasni`â€”ideal for AI agents that need to handle multiple tasks concurrently:

```python
import asyncio
from jasni import AsyncJasni

async def main():
    jasni = AsyncJasni("jsk_your_api_key")
    
    # Check agent's inbox
    emails = await jasni.emails.list(account="agent@yourdomain.com")
    for email in emails.emails:
        print(f"- {email.subject}")
    
    await jasni.close()

asyncio.run(main())
```

## Configuration

The SDK accepts optional configuration parameters:

```python
jasni = Jasni(
    "jsk_your_api_key",
    base_url="https://api.jasni.ai",  # Optional: custom base URL
    timeout=30.0,                      # Optional: request timeout in seconds
)
```

### Configuration Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `api_key` | `str` | Required | Your Jasni API key (starts with `jsk_`) |
| `base_url` | `str` | `https://api.jasni.ai` | API base URL |
| `timeout` | `float` | `30.0` | Request timeout in seconds |

## Type Hints

The SDK is fully typed with Python type hints and includes a `py.typed` marker for static type checkers:

```python
from jasni import Jasni
from jasni.types import (
    Email,
    Account,
    Label,
    Webhook,
    SendEmailResponse,
    ListEmailsResponse,
    Draft,
    EmailPreview,
)
```

## Context Manager Support

The SDK supports context managers for automatic resource cleanup:

```python
# Synchronous
with Jasni("jsk_your_api_key") as jasni:
    emails = jasni.emails.list(account="agent@yourdomain.com")

# Asynchronous
async with AsyncJasni("jsk_your_api_key") as jasni:
    emails = await jasni.emails.list(account="agent@yourdomain.com")
```

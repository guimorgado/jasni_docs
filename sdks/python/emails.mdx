The `emails` resource allows you to list, read, send, reply, forward, and manage emails.

## List Emails

Retrieve emails from a folder:

```python
result = jasni.emails.list(
    account="me@jasnimail.com",
    folder="INBOX",  # Optional: default is INBOX
    limit=50,        # Optional: default is 50, max 100
)

for email in result.emails:
    print(f"{email.from_} - {email.subject}")
```

### Options

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `account` | `str` | Yes | Email account to fetch from |
| `folder` | `str` | No | Folder to fetch from. Default: `INBOX` |
| `limit` | `int` | No | Number of emails to fetch. Default: 50, max: 100 |

### Response

```python
@dataclass
class EmailPreview:
    uid: int
    message_id: str
    subject: str
    from_: Union[EmailAddress, str]
    to: List[Union[EmailAddress, str]]
    date: str
    seen: bool
    has_attachments: bool
    snippet: Optional[str] = None

@dataclass
class ListEmailsResponse:
    emails: List[EmailPreview]
    folder: str
    total: int
    account: str
```

## Get Email

Retrieve a specific email by UID:

```python
result = jasni.emails.get(
    uid=123,
    account="me@jasnimail.com",
    folder="INBOX",  # Optional
)

print(f"Subject: {result.email.subject}")
print(f"From: {result.email.from_}")
print(f"Body: {result.email.text or result.email.html}")
```

### Options

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `uid` | `int` | Yes | Email UID |
| `account` | `str` | Yes | Email account |
| `folder` | `str` | No | Folder. Default: `INBOX` |

### Response

The `Email` type extends `EmailPreview` with full content:

```python
@dataclass
class Email(EmailPreview):
    text: Optional[str] = None
    html: Optional[str] = None
    cc: Optional[List[Union[EmailAddress, str]]] = None
    bcc: Optional[List[Union[EmailAddress, str]]] = None
    reply_to: Optional[str] = None
    attachments: Optional[List[EmailAttachment]] = None
    in_reply_to: Optional[str] = None
    references: Optional[str] = None
```

## Send Email

Send a new email:

```python
result = jasni.emails.send(
    from_="me@jasnimail.com",
    to=["recipient@example.com"],
    subject="Hello!",
    text="Plain text body",
    html="<p>HTML body</p>",      # Optional
    cc=["cc@example.com"],         # Optional
    bcc=["bcc@example.com"],       # Optional
    reply_to="reply@example.com",  # Optional
)

print(f"Message ID: {result.message_id}")
```

### Options

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from_` | `str` | Yes | Sender email (must be an account you own) |
| `to` | `str \| List[str]` | Yes | Recipient email address(es) |
| `subject` | `str` | Yes | Email subject |
| `text` | `str` | No | Plain text body |
| `html` | `str` | No | HTML body |
| `cc` | `str \| List[str]` | No | CC recipients |
| `bcc` | `str \| List[str]` | No | BCC recipients |
| `reply_to` | `str` | No | Reply-to address |

<Note>
  You must provide at least `text` or `html` for the email body.
</Note>

<Warning>
  **Attachments**: Sending attachments via the API is coming soon. See the [Attachments](/concepts/attachments) page for details on downloading attachments from received emails.
</Warning>

### Response

```python
@dataclass
class SendEmailResponse:
    message_id: str
    from_: str
    to: List[str]
    subject: str
    saved_to_sent: bool
```

## Reply to Email

Reply to an existing email:

```python
result = jasni.emails.reply(
    uid=123,
    account="me@jasnimail.com",
    text="Thanks for your email!",
    reply_all=False,         # Optional: default False
    include_original=True,   # Optional: default True
)

print(f"Reply sent: {result.message_id}")
```

### Options

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `uid` | `int` | Yes | Email UID to reply to |
| `account` | `str` | Yes | Email account |
| `folder` | `str` | No | Folder of the original email. Default: `INBOX` |
| `text` | `str` | No | Plain text reply body |
| `html` | `str` | No | HTML reply body |
| `reply_all` | `bool` | No | Reply to all recipients. Default: `False` |
| `include_original` | `bool` | No | Include original message. Default: `True` |

### Response

```python
@dataclass
class ReplyResponse:
    message_id: str
    from_: str
    to: List[str]
    cc: Optional[List[str]] = None
    subject: str
    in_reply_to: str
    references: str
    saved_to_sent: bool
```

## Forward Email

Forward an email to other recipients:

```python
result = jasni.emails.forward(
    uid=123,
    account="me@jasnimail.com",
    to=["forward@example.com"],
    text="FYI - see below",    # Optional message
    cc=["cc@example.com"],     # Optional
)

print(f"Forwarded: {result.message_id}")
```

### Options

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `uid` | `int` | Yes | Email UID to forward |
| `account` | `str` | Yes | Email account |
| `folder` | `str` | No | Folder of the original email. Default: `INBOX` |
| `to` | `str \| List[str]` | Yes | Recipients to forward to |
| `cc` | `str \| List[str]` | No | CC recipients |
| `bcc` | `str \| List[str]` | No | BCC recipients |
| `text` | `str` | No | Message to include before forwarded content |
| `html` | `str` | No | HTML message |
| `include_original` | `bool` | No | Include original message. Default: `True` |

### Response

```python
@dataclass
class ForwardResponse:
    message_id: str
    from_: str
    to: List[str]
    cc: Optional[List[str]] = None
    subject: str
    saved_to_sent: bool
    forwarded_from: ForwardedFrom
```

## Delete Email

Delete an email:

```python
jasni.emails.delete(uid=123, account="me@jasnimail.com")
```

## Mark as Read/Unread

Update the read status of an email:

```python
# Mark as read
jasni.emails.mark_as_read(uid=123, account="me@jasnimail.com")

# Mark as unread
jasni.emails.mark_as_unread(uid=123, account="me@jasnimail.com")
```

## Example: Email Workflow

Here's a complete example of an email workflow:

```python
from jasni import Jasni

jasni = Jasni("jsk_your_api_key")

def email_workflow():
    account = "me@jasnimail.com"
    
    # List recent emails
    result = jasni.emails.list(account=account, limit=10)
    print(f"Found {len(result.emails)} emails")
    
    # Get the first unread email
    unread = next((e for e in result.emails if not e.seen), None)
    if unread:
        # Get full content
        email_data = jasni.emails.get(uid=unread.uid, account=account)
        email = email_data.email
        print(f"Subject: {email.subject}")
        print(f"Body: {email.text}")
        
        # Mark as read
        jasni.emails.mark_as_read(uid=unread.uid, account=account)
        
        # Reply to it
        jasni.emails.reply(
            uid=unread.uid,
            account=account,
            text="Thank you for your email! I will get back to you soon.",
        )
    
    # Send a new email
    jasni.emails.send(
        from_=account,
        to="contact@example.com",
        subject="Hello from SDK",
        text="This is a test email sent from the Jasni Python SDK.",
        html="<p>This is a test email sent from the <strong>Jasni Python SDK</strong>.</p>",
    )

email_workflow()
```

## Async Example

```python
import asyncio
from jasni import AsyncJasni

async def email_workflow_async():
    jasni = AsyncJasni("jsk_your_api_key")
    account = "me@jasnimail.com"
    
    # List recent emails
    result = await jasni.emails.list(account=account, limit=10)
    print(f"Found {len(result.emails)} emails")
    
    # Send a new email
    await jasni.emails.send(
        from_=account,
        to="contact@example.com",
        subject="Hello from Async SDK",
        text="This is a test email sent asynchronously.",
    )
    
    await jasni.close()

asyncio.run(email_workflow_async())
```

---
title: "Accounts"
description: "Manage email accounts with the Jasni Python SDK"
---

The `accounts` resource allows you to create, list, and delete email accounts associated with your Jasni account.

## List Accounts

Retrieve all email accounts:

```python
result = jasni.accounts.list()

print(f"Found {result.total} accounts:")
for account in result.accounts:
    print(f"- {account.email} ({account.name})")
```

### Response

```python
@dataclass
class Account:
    id: str
    email: str
    name: str
    created_at: str

@dataclass
class ListAccountsResponse:
    accounts: List[Account]
    total: int
```

## Create Account

Create a new email account:

```python
# Create with auto-generated username
result = jasni.accounts.create(name="My Account")
print(f"Created: {result.account.email}")

# Create with specific username
result = jasni.accounts.create(
    username="support",
    name="Support Team"
)
# Result: support@mail.jasni.ai
```

### Options

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `username` | `str` | No | Local part of the email address. Auto-generated if not provided. |
| `name` | `str` | No | Display name for the account |

### Response

```python
@dataclass
class CreateAccountResponse:
    account: Account
    message: str
```

## Delete Account

Delete an email account:

```python
result = jasni.accounts.delete("support@mail.jasni.ai")
print(result["message"])  # "Account deleted successfully"
```

### Response

```python
{
    "deleted_email": "support@mail.jasni.ai",
    "message": "Account deleted successfully"
}
```

<Warning>
  Deleting an account is permanent. All emails and data associated with the account will be lost.
</Warning>

## Example: Account Management

Here's a complete example of managing accounts:

```python
from jasni import Jasni

jasni = Jasni("jsk_your_api_key")

def manage_accounts():
    # List existing accounts
    result = jasni.accounts.list()
    print("Current accounts:", [a.email for a in result.accounts])
    
    # Create a new account
    new_account = jasni.accounts.create(
        username="newsletter",
        name="Newsletter"
    )
    print(f"Created: {new_account.account.email}")
    
    # Delete an account
    jasni.accounts.delete(new_account.account.email)
    print(f"Deleted: {new_account.account.email}")

manage_accounts()
```

## Async Example

```python
import asyncio
from jasni import AsyncJasni

async def manage_accounts_async():
    jasni = AsyncJasni("jsk_your_api_key")
    
    # List existing accounts
    result = await jasni.accounts.list()
    print("Current accounts:", [a.email for a in result.accounts])
    
    # Create a new account
    new_account = await jasni.accounts.create(
        username="newsletter",
        name="Newsletter"
    )
    print(f"Created: {new_account.account.email}")
    
    # Delete an account
    await jasni.accounts.delete(new_account.account.email)
    print(f"Deleted: {new_account.account.email}")
    
    await jasni.close()

asyncio.run(manage_accounts_async())
```

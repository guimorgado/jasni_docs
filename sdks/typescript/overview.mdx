## Quick Start

```typescript
import { Jasni } from 'jasni-sdk'

// Initialize the client with your API key
const jasni = new Jasni('jsk_your_api_key')

// List your email accounts
const { accounts } = await jasni.accounts.list()
console.log(accounts)

// Send an email
const result = await jasni.emails.send({
  from: 'me@mail.jasni.ai',
  to: 'recipient@example.com',
  subject: 'Hello from Jasni!',
  text: 'This email was sent using the Jasni SDK.',
})
console.log('Sent:', result.messageId)
```

## Configuration

The SDK accepts optional configuration parameters:

```typescript
const jasni = new Jasni('jsk_your_api_key', {
  baseUrl: 'https://api.jasni.ai', // Optional: custom base URL
})
```

### Configuration Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `baseUrl` | `string` | `https://api.jasni.ai` | API base URL |

## TypeScript Support

The SDK is written in TypeScript and includes full type definitions. Import types as needed:

```typescript
import type { 
  Email, 
  Account, 
  Label, 
  Webhook,
  SendEmailOptions,
  ListEmailsOptions,
  Draft,
  EmailPreview,
} from 'jasni-sdk'
```

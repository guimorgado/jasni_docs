Messages are the core of Jasni. Every email you send, receive, reply to, or forward is a message. The API provides full control over message operations.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `uid` | integer | Unique identifier within the mailbox |
| `messageId` | string | RFC 2822 Message-ID header |
| `from` | string | Sender email address |
| `to` | string[] | Recipient email addresses |
| `cc` | string[] | CC recipients |
| `bcc` | string[] | BCC recipients (only visible to sender) |
| `subject` | string | Email subject line |
| `date` | string | When the email was sent/received (ISO 8601 format) |
| `text` | string | Plain text body |
| `html` | string | HTML body |
| `attachments` | object[] | List of attachments |
| `flags` | string[] | IMAP flags (e.g., `\Seen`, `\Flagged`) |

## Fetching Messages

Specify which inbox to use with the `account` parameter. You can also specify a `folder` (defaults to `INBOX`):

```bash
GET /api/v1/emails?account=support-agent@jasnimail.com
```

```bash
GET /api/v1/emails?account=sales-agent@jasnimail.com&folder=Sent
```

## SDK Usage

Manage messages programmatically using the Jasni SDKs.

<Tabs>
  <Tab title="TypeScript">
    **List Messages**
    
    Retrieve messages from an inbox:
    
    ```typescript
    import { Jasni } from 'jasni-sdk'
    
    const jasni = new Jasni('jsk_your_api_key')
    
    const { emails, total } = await jasni.emails.list({
      account: 'support-agent@jasnimail.com',
      folder: 'INBOX',
      limit: 20,
    })
    
    console.log(`Found ${total} messages:`)
    emails.forEach(email => {
      console.log(`- ${email.subject} from ${email.from}`)
    })
    ```
    
    **Get Message**
    
    Retrieve a specific message by UID:
    
    ```typescript
    const { email } = await jasni.emails.get(123, {
      account: 'support-agent@jasnimail.com',
    })
    console.log(email.subject, email.text)
    ```
    
    **Send Message**
    
    Send a new email:
    
    ```typescript
    const result = await jasni.emails.send({
      from: 'support-agent@jasnimail.com',
      to: 'customer@example.com',
      subject: 'Hello!',
      text: 'Plain text body',
      html: '<p>HTML body</p>',
    })
    console.log('Sent with ID:', result.messageId)
    ```
    
    | Parameter | Type | Required | Description |
    |-----------|------|----------|-------------|
    | `from` | `string` | Yes | Sender email address (must be an account you own) |
    | `to` | `string` or `string[]` | Yes | Recipient email address(es) |
    | `subject` | `string` | Yes | Email subject |
    | `text` | `string` | No | Plain text body |
    | `html` | `string` | No | HTML body |
    | `cc` | `string` or `string[]` | No | CC recipients |
    | `bcc` | `string` or `string[]` | No | BCC recipients |
    
    **Reply to Message**
    
    Reply to an existing email:
    
    ```typescript
    const result = await jasni.emails.reply(123, {
      account: 'support-agent@jasnimail.com',
      text: 'Thanks for your email!',
      replyAll: false,
    })
    ```
    
    **Forward Message**
    
    Forward an email to new recipients:
    
    ```typescript
    const result = await jasni.emails.forward(123, {
      account: 'support-agent@jasnimail.com',
      to: 'colleague@example.com',
      text: 'FYI - see the email below',
    })
    ```
    
    **Delete Message**
    
    Delete an email:
    
    ```typescript
    await jasni.emails.delete(123, {
      account: 'support-agent@jasnimail.com',
    })
    console.log('Message deleted')
    ```
    
    <Note>
      Deleting a message moves it to the Trash folder. It is not permanently deleted immediately.
    </Note>
    
  </Tab>
  
  <Tab title="Python">
    **List Messages**
    
    Retrieve messages from an inbox:
    
    ```python
    from jasni import Jasni
    
    jasni = Jasni("jsk_your_api_key")
    
    result = jasni.emails.list(
        account="support-agent@jasnimail.com",
        folder="INBOX",
        limit=20
    )
    
    print(f"Found {result.total} messages:")
    for email in result.emails:
        print(f"- {email.subject} from {email.from_}")
    ```
    
    **Get Message**
    
    Retrieve a specific message by UID:
    
    ```python
    result = jasni.emails.get(123, account="support-agent@jasnimail.com")
    print(result.email.subject, result.email.text)
    ```
    
    **Send Message**
    
    Send a new email:
    
    ```python
    result = jasni.emails.send(
        from_="support-agent@jasnimail.com",
        to="customer@example.com",
        subject="Hello!",
        text="Plain text body",
        html="<p>HTML body</p>"
    )
    print(f"Sent with ID: {result.message_id}")
    ```
    
    | Parameter | Type | Required | Description |
    |-----------|------|----------|-------------|
    | `from_` | `str` | Yes | Sender email address (must be an account you own) |
    | `to` | `str` or `list[str]` | Yes | Recipient email address(es) |
    | `subject` | `str` | Yes | Email subject |
    | `text` | `str` | No | Plain text body |
    | `html` | `str` | No | HTML body |
    | `cc` | `str` or `list[str]` | No | CC recipients |
    | `bcc` | `str` or `list[str]` | No | BCC recipients |
    
    **Reply to Message**
    
    Reply to an existing email:
    
    ```python
    result = jasni.emails.reply(
        123,
        account="support-agent@jasnimail.com",
        text="Thanks for your email!",
        reply_all=False
    )
    ```
    
    **Forward Message**
    
    Forward an email to new recipients:
    
    ```python
    result = jasni.emails.forward(
        123,
        account="support-agent@jasnimail.com",
        to="colleague@example.com",
        text="FYI - see the email below"
    )
    ```
    
    **Delete Message**
    
    Delete an email:
    
    ```python
    jasni.emails.delete(123, account="support-agent@jasnimail.com")
    print("Message deleted")
    ```
    
    <Note>
      Deleting a message moves it to the Trash folder. It is not permanently deleted immediately.
    </Note>
  </Tab>

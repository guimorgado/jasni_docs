An inbox represents an email account for your AI agent. Each inbox has its own email address and can send, receive, and manage emails independently.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `id` | string | Unique identifier for the inbox |
| `email` | string | The agent's email address |
| `name` | string | Display name (e.g., "Support Agent", "Booking Assistant") |
| `created_at` | string | When the inbox was created (ISO 8601 format) |

## Fetching Emails

Specify which inbox to use with the `account` parameter. You can also specify a `folder` (defaults to `INBOX`):

```bash
GET /api/v1/emails?account=support-agent@jasnimail.com
```

```bash
GET /api/v1/emails?account=sales-agent@jasnimail.com&folder=Sent
```

## SDK Usage

Manage inboxes programmatically using the Jasni SDKs.

<Tabs>
  <Tab title="TypeScript">
    **List Inboxes**
    
    Retrieve all inboxes associated with your account:
    
    ```typescript
    import { Jasni } from 'jasni-sdk'
    
    const jasni = new Jasni('jsk_your_api_key')
    
    const { accounts, total } = await jasni.accounts.list()
    
    console.log(`Found ${total} inboxes:`)
    accounts.forEach(account => {
      console.log(`- ${account.email} (${account.name})`)
    })
    ```
    
    **Create Inbox**
    
    Create a new inbox for your AI agent:
    
    ```typescript
    // Create with auto-generated username
    const { account } = await jasni.accounts.create({
      name: 'Support Agent',
    })
    console.log('Created inbox:', account.email)
    
    // Create with specific username
    const { account: salesAgent } = await jasni.accounts.create({
      username: 'sales-agent',
      name: 'Sales Agent',
    })
    // Result: sales-agent@jasnimail.com
    ```
    
    | Parameter | Type | Required | Description |
    |-----------|------|----------|-------------|
    | `username` | `string` | No | Local part of the email address. Auto-generated if not provided. |
    | `name` | `string` | No | Display name for the inbox |
    
    **Update Inbox**
    
    Update an existing inbox's display name:
    
    ```typescript
    const { account } = await jasni.accounts.update({
      email: 'sales-agent@jasnimail.com',
      name: 'Enterprise Sales Agent',
    })
    console.log('Updated:', account.name)
    ```
    
    **Delete Inbox**
    
    Delete an inbox permanently:
    
    ```typescript
    await jasni.accounts.delete('sales-agent@jasnimail.com')
    console.log('Inbox deleted')
    ```
    
    <Warning>
      Deleting an inbox is permanent. All emails and data associated with it will be lost.
    </Warning>
    
  </Tab>
  
  <Tab title="Python">
    **List Inboxes**
    
    Retrieve all inboxes associated with your account:
    
    ```python
    from jasni import Jasni
    
    jasni = Jasni("jsk_your_api_key")
    
    result = jasni.accounts.list()
    
    print(f"Found {result.total} inboxes:")
    for account in result.accounts:
        print(f"- {account.email} ({account.name})")
    ```
    
    **Create Inbox**
    
    Create a new inbox for your AI agent:
    
    ```python
    # Create with auto-generated username
    result = jasni.accounts.create(name="Support Agent")
    print(f"Created inbox: {result.account.email}")
    
    # Create with specific username
    result = jasni.accounts.create(
        username="sales-agent",
        name="Sales Agent"
    )
    # Result: sales-agent@jasnimail.com
    ```
    
    | Parameter | Type | Required | Description |
    |-----------|------|----------|-------------|
    | `username` | `str` | No | Local part of the email address. Auto-generated if not provided. |
    | `name` | `str` | No | Display name for the inbox |
    
    **Update Inbox**
    
    Update an existing inbox's display name:
    
    ```python
    result = jasni.accounts.update(
        "sales-agent@jasnimail.com",
        name="Enterprise Sales Agent"
    )
    print(f"Updated: {result.account.name}")
    ```
    
    **Delete Inbox**
    
    Delete an inbox permanently:
    
    ```python
    jasni.accounts.delete("sales-agent@jasnimail.com")
    print("Inbox deleted")
    ```
    
    <Warning>
      Deleting an inbox is permanent. All emails and data associated with it will be lost.
    </Warning>

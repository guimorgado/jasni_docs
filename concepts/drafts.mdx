Drafts allow you to save email messages before sending them. This is useful for composing emails over time, scheduling sends, or building approval workflows where AI agents compose responses for human review.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `uid` | integer | Unique identifier within the mailbox |
| `from` | string | Sender email address |
| `to` | string[] | Recipient email addresses |
| `cc` | string[] | CC recipients |
| `bcc` | string[] | BCC recipients |
| `subject` | string | Email subject line |
| `text` | string | Plain text body |
| `html` | string | HTML body |
| `attachments` | object[] | List of attachments |

## Fetching Drafts

List drafts for an inbox with the `account` parameter:

```bash
GET /api/v1/emails/drafts?account=support-agent@jasnimail.com
```

```bash
GET /api/v1/emails/drafts?account=sales-agent@jasnimail.com&limit=50
```

## SDK Usage

Manage drafts programmatically using the Jasni SDKs.

<Tabs>
  <Tab title="TypeScript">
    **List Drafts**
    
    Retrieve all drafts from an inbox:
    
    ```typescript
    import { Jasni } from 'jasni-sdk'
    
    const jasni = new Jasni('jsk_your_api_key')
    
    const { drafts, total } = await jasni.drafts.list({
      account: 'support-agent@jasnimail.com',
      limit: 50,
    })
    
    console.log(`Found ${total} drafts:`)
    drafts.forEach(draft => {
      console.log(`- ${draft.subject}`)
    })
    ```
    
    **Get Draft**
    
    Retrieve a specific draft by UID:
    
    ```typescript
    const { email } = await jasni.drafts.get(123, {
      account: 'support-agent@jasnimail.com',
    })
    console.log(email.subject, email.text)
    ```
    
    **Create Draft**
    
    Save a new draft email:
    
    ```typescript
    const { uid } = await jasni.drafts.create({
      account: 'support-agent@jasnimail.com',
      to: 'customer@example.com',
      subject: 'Work in progress',
      text: 'This is a draft I am working on...',
      html: '<p>This is a draft I am working on...</p>',
    })
    console.log('Created draft with UID:', uid)
    ```
    
    | Parameter | Type | Required | Description |
    |-----------|------|----------|-------------|
    | `account` | `string` | Yes | Email account to save the draft in |
    | `to` | `string \| string[]` | No | Recipient email address(es) |
    | `subject` | `string` | No | Email subject |
    | `text` | `string` | No | Plain text body |
    | `html` | `string` | No | HTML body |
    | `cc` | `string \| string[]` | No | CC recipients |
    
    **Update Draft**
    
    Modify an existing draft:
    
    ```typescript
    const { uid } = await jasni.drafts.update({
      account: 'support-agent@jasnimail.com',
      uid: 123,
      subject: 'Updated subject',
      text: 'Updated content...',
    })
    console.log('Updated draft:', uid)
    ```
    
    **Send Draft**
    
    Convert a draft into a sent email:
    
    ```typescript
    const result = await jasni.drafts.send({
      account: 'support-agent@jasnimail.com',
      uid: 123,
    })
    console.log('Sent with message ID:', result.messageId)
    ```
    
    <Note>
      When you send a draft, it is automatically moved from the Drafts folder to Sent.
    </Note>
    
    **Delete Draft**
    
    Remove a draft without sending:
    
    ```typescript
    await jasni.drafts.delete({
      account: 'support-agent@jasnimail.com',
      uid: 123,
    })
    console.log('Draft deleted')
    ```

  <Tab title="Python">
    **List Drafts**
    
    Retrieve all drafts from an inbox:
    
    ```python
    from jasni import Jasni
    
    jasni = Jasni("jsk_your_api_key")
    
    result = jasni.drafts.list(
        account="support-agent@jasnimail.com",
        limit=50
    )
    
    print(f"Found {result.total} drafts:")
    for draft in result.drafts:
        print(f"- {draft.subject}")
    ```
    
    **Get Draft**
    
    Retrieve a specific draft by UID:
    
    ```python
    result = jasni.drafts.get(123, account="support-agent@jasnimail.com")
    print(result.email.subject, result.email.text)
    ```
    
    **Create Draft**
    
    Save a new draft email:
    
    ```python
    result = jasni.drafts.create(
        account="support-agent@jasnimail.com",
        to="customer@example.com",
        subject="Work in progress",
        text="This is a draft I am working on...",
        html="<p>This is a draft I am working on...</p>"
    )
    print(f"Created draft with UID: {result.uid}")
    ```
    
    | Parameter | Type | Required | Description |
    |-----------|------|----------|-------------|
    | `account` | `str` | Yes | Email account to save the draft in |
    | `to` | `str \| list[str]` | No | Recipient email address(es) |
    | `subject` | `str` | No | Email subject |
    | `text` | `str` | No | Plain text body |
    | `html` | `str` | No | HTML body |
    | `cc` | `str \| list[str]` | No | CC recipients |
    
    **Update Draft**
    
    Modify an existing draft:
    
    ```python
    result = jasni.drafts.update(
        account="support-agent@jasnimail.com",
        uid=123,
        subject="Updated subject",
        text="Updated content..."
    )
    print(f"Updated draft: {result.uid}")
    ```
    
    **Send Draft**
    
    Convert a draft into a sent email:
    
    ```python
    result = jasni.drafts.send(
        account="support-agent@jasnimail.com",
        uid=123
    )
    print(f"Sent with message ID: {result.message_id}")
    ```
    
    <Note>
      When you send a draft, it is automatically moved from the Drafts folder to Sent.
    </Note>
    
    **Delete Draft**
    
    Remove a draft without sending:
    
    ```python
    jasni.drafts.delete(account="support-agent@jasnimail.com", uid=123)
    ```

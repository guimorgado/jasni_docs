Attachments allow you to receive and download files from your emails. Jasni supports all common file types and provides metadata about attachments when listing or reading emails.

<Warning>
  **Sending Attachments**: Sending attachments via the API is coming soon. Currently, the send endpoint only supports text and HTML bodies. The examples below show the planned interface.
</Warning>

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `filename` | string | Original filename |
| `contentType` | string | MIME type (e.g., `application/pdf`) |
| `size` | integer | File size in bytes |
| `contentId` | string | Content-ID for inline attachments |
| `content` | string | Base64 encoded file content (when fetched) |

## Fetching Attachments

Get all attachments from an email:

```bash
GET /api/v1/emails/123/attachments?account=support-agent@jasnimail.com
```

Get a specific attachment by filename or index:

```bash
GET /api/v1/emails/123/attachments/document.pdf?account=support-agent@jasnimail.com
```

```bash
GET /api/v1/emails/123/attachments/0?account=support-agent@jasnimail.com
```

## SDK Usage

Manage attachments programmatically using the Jasni SDKs.

<Tabs>
  <Tab title="TypeScript">
    **Get All Attachments**
    
    Retrieve all attachments from an email with their content:
    
    ```typescript
    import { Jasni } from 'jasni-sdk'
    
    const jasni = new Jasni('jsk_your_api_key')
    
    const { attachments } = await jasni.emails.getAttachments(123, {
      account: 'support-agent@jasnimail.com',
    })
    
    for (const att of attachments) {
      console.log(`${att.filename}: ${att.contentType}, ${att.size} bytes`)
      // att.content is base64 encoded
    }
    ```
    
    **Get Specific Attachment**
    
    Retrieve a single attachment by filename or index:
    
    ```typescript
    // Get by filename
    const { attachment } = await jasni.emails.getAttachment(123, {
      account: 'support-agent@jasnimail.com',
      attachment: 'document.pdf',
    })
    
    // Get by index (0-based)
    const { attachment: first } = await jasni.emails.getAttachment(123, {
      account: 'support-agent@jasnimail.com',
      attachment: 0,
    })
    
    console.log(attachment.filename, attachment.size)
    ```
    
    **Send Email with Attachments**
    
    Send an email with file attachments:
    
    ```typescript
    import { readFileSync } from 'fs'
    
    // Read file and convert to base64
    const fileBuffer = readFileSync('./document.pdf')
    const base64Content = fileBuffer.toString('base64')
    
    await jasni.emails.send({
      from: 'support-agent@jasnimail.com',
      to: 'customer@example.com',
      subject: 'Document attached',
      text: 'Please find the document attached.',
      attachments: [
        {
          filename: 'document.pdf',
          content: base64Content,
          contentType: 'application/pdf',
        },
      ],
    })
    ```
    
    | Parameter | Type | Required | Description |
    |-----------|------|----------|-------------|
    | `filename` | `string` | Yes | Name of the file |
    | `content` | `string` | Yes | Base64 encoded file content |
    | `contentType` | `string` | Yes | MIME type of the file |
    | `contentId` | `string` | No | Content-ID for inline attachments |
    
    **Inline Attachments**
    
    Embed images in HTML emails:
    
    ```typescript
    await jasni.emails.send({
      from: 'support-agent@jasnimail.com',
      to: 'customer@example.com',
      subject: 'Welcome!',
      html: '<p>See our logo:</p><img src="cid:logo123">',
      attachments: [
        {
          filename: 'logo.png',
          content: logoBase64,
          contentType: 'image/png',
          contentId: 'logo123',
        },
      ],
    })
    ```

  <Tab title="Python">
    **Get All Attachments**
    
    Retrieve all attachments from an email with their content:
    
    ```python
    from jasni import Jasni
    
    jasni = Jasni("jsk_your_api_key")
    
    result = jasni.emails.get_attachments(
        123,
        account="support-agent@jasnimail.com"
    )
    
    for att in result.attachments:
        print(f"{att.filename}: {att.content_type}, {att.size} bytes")
        # att.content is base64 encoded
    ```
    
    **Get Specific Attachment**
    
    Retrieve a single attachment by filename or index:
    
    ```python
    # Get by filename
    result = jasni.emails.get_attachment(
        123,
        account="support-agent@jasnimail.com",
        attachment="document.pdf"
    )
    
    # Get by index (0-based)
    result = jasni.emails.get_attachment(
        123,
        account="support-agent@jasnimail.com",
        attachment=0
    )
    
    print(result.attachment.filename, result.attachment.size)
    ```
    
    **Send Email with Attachments**
    
    Send an email with file attachments:
    
    ```python
    import base64
    
    # Read file and convert to base64
    with open("document.pdf", "rb") as f:
        content = base64.b64encode(f.read()).decode("utf-8")
    
    jasni.emails.send(
        from_="support-agent@jasnimail.com",
        to="customer@example.com",
        subject="Document attached",
        text="Please find the document attached.",
        attachments=[
            {
                "filename": "document.pdf",
                "content": content,
                "contentType": "application/pdf",
            },
        ],
    )
    ```
    
    | Parameter | Type | Required | Description |
    |-----------|------|----------|-------------|
    | `filename` | `str` | Yes | Name of the file |
    | `content` | `str` | Yes | Base64 encoded file content |
    | `contentType` | `str` | Yes | MIME type of the file |
    | `contentId` | `str` | No | Content-ID for inline attachments |

## Size Limits

| Limit | Value |
|-------|-------|
| Single attachment | 25 MB |
| Total per email | 25 MB |

<Note>
  For larger files, consider using a file sharing service and including a link in the email.
</Note>

Labels allow you to categorize and organize emails. Unlike folders, an email can have multiple labels, making it easy to classify emails by topic, priority, or any custom criteria. Labels are particularly useful for AI agents that need to classify and route incoming emails.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `id` | string | Unique identifier (UUID) |
| `name` | string | Label name (1-50 characters) |
| `color` | string | Hex color code (e.g., `#6366f1`) |
| `description` | string | Optional description |
| `created_at` | string | Creation timestamp (ISO 8601) |
| `email_count` | integer | Number of emails with this label (when requested) |

## Fetching Labels

List all labels for your account:

```bash
GET /api/v1/labels
```

```bash
GET /api/v1/labels?include_count=true
```

## SDK Usage

Manage labels programmatically using the Jasni SDKs.

<Tabs>
  <Tab title="TypeScript">
    **List Labels**
    
    Retrieve all labels:
    
    ```typescript
    import { Jasni } from 'jasni-sdk'
    
    const jasni = new Jasni('jsk_your_api_key')
    
    // List labels
    const { labels } = await jasni.labels.list()
    
    // List labels with email counts
    const { labels: labelsWithCount } = await jasni.labels.list({
      includeCount: true,
    })
    
    labelsWithCount.forEach(label => {
      console.log(`- ${label.name}: ${label.emailCount} emails`)
    })
    ```
    
    **Create Label**
    
    Create a new label:
    
    ```typescript
    const { label } = await jasni.labels.create({
      name: 'Important',
      color: '#ef4444',
      description: 'High priority emails',
    })
    console.log('Created label:', label.id)
    ```
    
    | Parameter | Type | Required | Description |
    |-----------|------|----------|-------------|
    | `name` | `string` | Yes | Label name (1-50 characters) |
    | `color` | `string` | No | Hex color code. Auto-generated if not provided. |
    | `description` | `string` | No | Label description |
    
    **Delete Label**
    
    Delete a label:
    
    ```typescript
    const result = await jasni.labels.delete('label-id-123')
    console.log(result.message) // "Label deleted successfully"
    ```
    
    <Warning>
      Deleting a label also removes all email-label associations for that label.
    </Warning>
    
    **Assign Labels to Email**
    
    Assign labels to an email:
    
    ```typescript
    // Assign by label names (creates labels if they don't exist)
    await jasni.emails.labels.assign(123, {
      account: 'support-agent@jasnimail.com',
      labels: ['Important', 'Support'],
      agentName: 'my-classifier',
    })
    
    // Assign by label IDs
    await jasni.emails.labels.assign(123, {
      account: 'support-agent@jasnimail.com',
      labelIds: ['label-id-1', 'label-id-2'],
    })
    ```
    
    **Get Email Labels**
    
    Get all labels assigned to an email:
    
    ```typescript
    const { labels } = await jasni.emails.labels.list(123, {
      account: 'support-agent@jasnimail.com',
    })
    labels.forEach(label => {
      console.log(`- ${label.name}: ${label.color}`)
    })
    ```
    
    **Remove Labels from Email**
    
    Remove labels from an email:
    
    ```typescript
    await jasni.emails.labels.remove(123, {
      account: 'support-agent@jasnimail.com',
      labelIds: ['label-id-1'],
    })
    ```

  <Tab title="Python">
    **List Labels**
    
    Retrieve all labels:
    
    ```python
    from jasni import Jasni
    
    jasni = Jasni("jsk_your_api_key")
    
    # List labels
    result = jasni.labels.list()
    
    # List labels with email counts
    result = jasni.labels.list(include_count=True)
    
    for label in result.labels:
        print(f"- {label.name}: {label.email_count} emails")
    ```
    
    **Create Label**
    
    Create a new label:
    
    ```python
    result = jasni.labels.create(
        name="Important",
        color="#ef4444",
        description="High priority emails"
    )
    print(f"Created label: {result.label.id}")
    ```
    
    | Parameter | Type | Required | Description |

    | `color` | `str` | No | Hex color code. Auto-generated if not provided. |
